# Banking-System

A banking application built with Python and SQLite demonstrating OOP principles 
and database design.

## Features

- Customer and account management
- Deposit, withdraw, and transfer operations
- Real-time balance tracking
- Complete transaction history
- Input validation and error handling

## Tech Stack

- **Language**: Python 3.8+
- **Database**: SQLite3
- **Architecture**: Object-Oriented Programming (OOP)

## How to Run
```bash
python banking_system.py
```

## Database Design

Three interconnected tables:
- **customers**: Store customer information
- **accounts**: Store account details (linked to customers)
- **transactions**: Record all financial operations

## Key OOP Concepts

- **Encapsulation**: Data protection with private methods
- **Abstraction**: Complex logic hidden behind simple interfaces
- **Inheritance**: Reusable code structure

## What You Learn

- OOP principles and design patterns
- Relational database schema design
- Real-world business logic implementation
- Exception handling and validation
- Clean code architecture
